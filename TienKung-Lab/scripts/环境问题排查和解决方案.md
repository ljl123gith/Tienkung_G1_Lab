# 环境问题排查和解决方案

## 问题诊断

根据诊断结果，`isaaclab_new` 环境中存在以下问题：

### ✅ 已正确配置
- Python 版本: 3.10.19 ✓
- Conda 环境: isaaclab_new ✓  
- 基础依赖: torch, numpy, gym ✓

### ❌ 缺失的组件
- **Isaac Lab**: 未安装（核心问题）
- **legged_lab**: 未安装（项目模块）
- **rsl_rl**: 未安装（项目模块）

## 解决方案

您有两个选择：

### 方案 1: 在已安装 Isaac Lab 的环境中安装项目依赖（推荐）

您已经有 `env_isaaclab` 和 `env_isaaclab_v2` 环境安装了 Isaac Lab，可以直接使用：

```bash
# 激活已有 Isaac Lab 的环境
conda activate env_isaaclab  # 或 env_isaaclab_v2

# 安装项目依赖
cd /home/ljl/ljl_for_RL/G1DWAQ_Lab/TienKung-Lab
pip install -e .

# 安装 rsl_rl
cd rsl_rl
pip install -e .

# 返回项目目录
cd ..

# 验证安装
python scripts/check_environment.py

# 运行训练
python legged_lab/scripts/train.py --task=g1_dwaq --headless --num_envs=4096 --max_iterations=10000
```

### 方案 2: 在 isaaclab_new 环境中安装 Isaac Lab

如果您想使用新的 `isaaclab_new` 环境，需要先安装 Isaac Lab：

#### 步骤 1: 安装 Isaac Lab

Isaac Lab 不能通过 pip 直接安装，需要按照官方指南安装：

1. **访问官方安装指南**:
   https://isaac-sim.github.io/IsaacLab/main/source/setup/installation/index.html

2. **克隆 Isaac Lab 仓库** (如果还没有):
   ```bash
   # 选择一个位置克隆（不要在 G1DWAQ_Lab 目录内）
   cd ~
   git clone https://github.com/isaac-sim/IsaacLab.git
   cd IsaacLab
   ```

3. **按照官方文档安装**:
   - 使用 conda 安装方式（推荐）
   - 确保在 `isaaclab_new` 环境中安装
   - 需要 NVIDIA GPU 和 CUDA 支持
   - 需要安装 Isaac Sim（Isaac Lab 的依赖）

#### 步骤 2: 安装项目依赖

安装完 Isaac Lab 后：

```bash
# 确保在 isaaclab_new 环境中
conda activate isaaclab_new

# 安装项目依赖
cd /home/ljl/ljl_for_RL/G1DWAQ_Lab/TienKung-Lab
pip install -e .

# 安装 rsl_rl
cd rsl_rl
pip install -e .

# 返回项目目录
cd ..
```

#### 步骤 3: 验证安装

```bash
# 运行诊断脚本
python scripts/check_environment.py

# 如果所有检查通过，尝试运行训练
python legged_lab/scripts/train.py --task=g1_dwaq --headless --num_envs=64
```

## 常见问题

### Q: 为什么不能直接 `pip install isaaclab`？

A: Isaac Lab 是一个复杂的仿真框架，依赖于 NVIDIA Isaac Sim，需要按照官方指南进行完整安装，不能简单地通过 pip 安装。

### Q: 我应该使用哪个环境？

A: 如果您已经有 `env_isaaclab` 或 `env_isaaclab_v2` 且它们工作正常，建议使用这些环境（方案1）。如果 `isaaclab_new` 是您专门为这个项目创建的新环境，则需要在其中安装 Isaac Lab（方案2）。

### Q: 如何检查 Isaac Lab 是否正确安装？

A: 运行以下命令：
```bash
python -c "from isaaclab.app import AppLauncher; print('Isaac Lab 安装成功')"
```
